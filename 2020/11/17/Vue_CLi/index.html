环境的配置 （开发/测试/生产）

Vue-Cli官网地址

在Vue项目中 package.json

修改为

官网有说明这些参数的含义，但使用方式是在启动时在命令行内输入，不是很方便开发，
将参数写入package.json中就可以直接使用了
–open 启动时打开浏览器
–copy 服务器启动时将 URL复制到剪切板
–mode 指定环境模式 (默认值：development)
–host 指定 host (默认值：0.0.0.0)
–port 指定 port (默认值：8080)
–https 使用 https (默认值：false)

然后在项目的根目录下创建对应的 ‘.env’ 文件,配置不同模式下的环境变量
环境变量和模式

例如： “.env.dev” 内容如下

    NODE_ENV = 'development'
    VUE_APP_CURRENTMODE = 'dev'
    VUE_APP_BASEURL = 'https://dev.app.com'

VUE_APP_BASEURL 就是我们在 dev 开发模式下调用的接口地址域名了，test地址同理

然后我们就可以通过 process 变量来获取当前环境下的 url 地址
process 变量说明

注意：变量名必须以 "VUE_APP_ " 开头

个人觉得 相比创建 config.js 文件配置地址，这种配置方式更加安全，修改地址后必须重新启动项目才能生效避免了一些误操作
